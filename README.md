# Flashcards Web App

A simple flashcard app built with HTML, CSS, and JavaScript.

## Features

- Add flashcards one at a time
- Organize cards into named decks
- Review one deck at a time
- Export current deck to JSON
- Import cards and decks from JSON
- Mobile-friendly responsive design

## Usage

1. Open `index.html` in a browser
2. Add a flashcard using the form
3. Review cards using the Review mode
4. Export/import cards to move them between devices

## GitHub Pages

To host on GitHub Pages:

1. Push this repository to GitHub
2. Enable GitHub Pages from the repository settings
3. Set the source to the `main` branch or `docs` folder

Then open the provided Pages URL.
